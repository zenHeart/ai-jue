## What is ai-jue-core?

`ai-jue-core` is the engine room of the `ai-jue` framework. It provides the essential logic for:

- **Config Loading**: Parsing and validating `ai.config.js`.
- **Preset Management**: Loading and resolving remote or local presets.
- **Asset Orchestration**: Merging prompts and skills with "Intelligent Coexistence" strategy.
- **Adapter Integration**: Providing the interface and orchestration for tool-specific adapters.
