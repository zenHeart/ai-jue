## Features

Tailored AI interaction strategies for modern React development:

- **Hooks Design**: Best practices for custom hooks and state management.
- **Component Patterns**: Guidance for atomic design and component extraction.
- **Performance**: Strategies for minimizing re-renders and optimizing bundles.
- **Testing**: Patterns for React Testing Library and component unit tests.

## Installation

```bash
npm install jue-preset-react
```

## Usage

```javascript
// ai.config.js
export default {
  preset: 'react'
}
```
