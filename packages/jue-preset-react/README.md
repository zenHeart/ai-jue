## 功能描述

为现代 React 开发量身定制的 AI 系统指令：

- **Hooks 设计**：自定义 Hooks 与状态管理的最佳实践。
- **组件模式**：原子设计与组件提炼指引。
- **性能优化**：减少重渲染与包体积优化的 AI 策略。
- **测试规范**：React Testing Library 与组件单元测试模式。

## 安装

```bash
npm install jue-preset-react
```

## 使用

```javascript
// ai.config.js
export default {
  preset: 'react'
}
```
