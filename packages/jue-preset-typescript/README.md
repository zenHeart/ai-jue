## 功能描述

增强 AI 对 TypeScript 工程化的理解能力：

- **类型安全**：鲁棒的类型定义与避免 `any` 的最佳实践。
- **泛型技巧**：高级泛型设计与约束模式。
- **工具类型**：提取与转换类型的 AI 指引。
- **工程配置**：标准化的 `tsconfig` 与构建工具对齐策略。

## 安装

```bash
npm install jue-preset-typescript
```

## 使用

```javascript
// ai.config.js
export default {
  presets: ['base', 'typescript']
}
```
