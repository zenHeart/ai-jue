## Features

Enhancing AI's understanding of TypeScript engineering:

- **Type Safety**: Patterns for robust typing and avoiding `any`.
- **Generics**: Advanced generic design and constraints.
- **Utility Types**: Best practices for extracting and transforming types.
- **Configuration**: Standardized `tsconfig` and build tool alignment.

## Installation

```bash
npm install jue-preset-typescript
```

## Usage

```javascript
// ai.config.js
export default {
  presets: ['base', 'typescript']
}
```
